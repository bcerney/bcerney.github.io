Homelab 1.1

August 1, 2020










Throughout the summer I’ve made a few homelab updates:

  • Replaced the Raspberry Pi 3 Model B+ with a RPi 3 Model B running some Docker containers: Pi-hole, and a UniFi controller for my UniFi UAP-AC-Lite. I plan to repurpose the RPi 3 B+ for some sort of portable gadget project in the future.
  • Added a RPi 4 Model B 4GB running NextCloudPi via Docker for local cloud storage. I’ve also attached a WD Elements 2TB hard drive to increase storage capacity and reduce R/W of the RPi’s microSD.

If your interested in the setup process, I’ve compiled my notes in a GitHub repository. I mounted the RPi’s in a cluster case, with intentions of adding to it in the future.
